# Task Manager Android

A modern Android Task Management application built using Kotlin and Jetpack Compose.

## Features

* User Authentication
* Create Tasks
* Search Tasks
* Task Priorities
* Due Dates
* Navigation Drawer
* Bottom Sheet Task Creation
* Spring Boot Backend Integration

## Tech Stack

### Frontend

* Kotlin
* Jetpack Compose
* Material 3

### Backend

* Spring Boot
* MySQL
* JWT Authentication
